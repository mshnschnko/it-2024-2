# Приложение "Список дел"

1. Use Cases

- Гость
    - Регистрация
    - Вход
- Пользователь
    - Выход
    - Просмотр списка дел
    - Добавление дела
    - Изменение дела
    - Удаление дела

2. Пользовательский интерфейс

- Страницы
    - Вход (email, password)
    - Регистрация (email, password, password)
    - Список дел (таблица)
    - Редактор дел (форма) - (название, дата окончания, важность (check))

3. Требования
    - динамическое переключение языка на всех страницах (t9n)

4. Технологии
    - java script (redom0)
    - bootstrap
